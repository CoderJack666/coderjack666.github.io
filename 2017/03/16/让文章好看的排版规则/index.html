让文章好看的排版规则

发表于 2017-03-16 | 分类于 Ideas | 阅读次数

一篇好的文章不仅在于内容的精彩，一个好的排版也会让人心旷神怡、兴趣盎然。

空格

空格是字间的留白，让词、句显得轻快、明朗。

中英文之间加空格

e.g. 使用 IDE 能提高 Java 开发的效率。

中文与数字之间加空格

e.g. 不知不觉，又到了阳春 3 月。

数字与单位之间加空格

e.g. 一个 20 TB 的硬盘能装多少电影。

完整的英文整句标点与单词之间加空格

e.g. Stay hungry, stay foolish. You can do better.

度、百分号不加空格

e.g. 武汉夏天的气温能到 40° 。

e.g. 今天的空气湿度是 82% 。

全角标点与其他字符之间不加空格

e.g. 学习 Java，你会感到快乐。

标点

标点符号分为「全角」和「半角」两种，「全角」标点占两个字节，「半角」符号占一个字节。

使用全角中文标点

中文排版中所有的标点都应该使用中文全角标点

e.g. 推荐你学习 Java。

遇到英文整句、特殊名词时使用半角标点

e.g. 乔布斯说过：「Stay hungry, stay foolish.」

e.g. Facebook, Inc.

使用直角引号

e.g. 大家都来用「方引号」！

引号中使用引号

e.g. 问你一个问题：「你知道『方引号』是啥不？」

其他

英文名词首字母大写

e.g. Google、Android、Facebook。

专有名词使用正确的大小写

e.g. GitHub、iOS、iPhone 6S、MacBook Pro。

首行不要缩进

使用空一行的形式区分自然段，这样更自然、更方便。

适当的间距、空行

如果段落太长，就可以适当的空行分段，这样不会给读者压力，提升阅读体验。
